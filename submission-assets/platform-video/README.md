# EcoGenesis workflow-first platform demo video

Russian contest/demo walkthrough with voiceover, burned subtitles and full-page scrolling captures.

The walkthrough starts from the `Workflow` screen, then moves through `Overview`,
`Run Compiler`, `Evidence Map`, `Fragment Graph`, `Validation`,
`Methods & Audits` and `Evidence Pack`.

The source captures live in `screenshots-scroll/` so the video shows more than the top of each page.

Build:

```bash
submission-assets/platform-video/build_platform_video_ru.sh
```

- `watch-local.html`
- `video/ecogenesis-platform-demo-ru-voice-subs.mp4`
- `video/ecogenesis-platform-demo-ru-voice-subs.srt`
- `video/ecogenesis-platform-demo-ru-voiceover.txt`
